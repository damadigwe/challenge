
How to run number-guessing game
------------------------------------


To run program :

Install JDK 6.x if you don't already have it

Intsall Maven 4.x if you don't already have it

Set Java PATH as well as Maven PATH ( Please google details on how to do this, in case you need to)

Copy project folder(from github.com) to any folder in your operating system

Change directory to project root folder, from the command prompt - ie directory with the pom.xml file

Type : "mvn compile", and hit enter key to compile project files 

Type : "mvn exec:java  -Dexec.mainClass=com.interview.game.GameDriver" and  hit enter at the command line

Follow operational instruction manual that appears at the console to continue

Alternatively, import project files into Eclipse IDE and run from Eclipse
